# Library Book Tracker

A React application to track books you have read, are currently reading, and plan to read.

## Features

- **Search and Add Books**: Search for books by title or author using the Open Library API.
- **Track Reading Status**: Move books between "Plan to Read", "Currently Reading", and "Read" lists.
- **Persistence**: Your library is saved locally in your browser.
- **Modern Design**: Styled with Tailwind CSS.

## Live Demo

Check out the live application here: [https://erdogduarda.github.io/react-library/](https://erdogduarda.github.io/react-library/)

- React

- Tailwind CSS
